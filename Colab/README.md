# Google Colaboratory Notebooks

## Overview

This directory contains interactive Jupyter notebooks designed specifically for Google Colaboratory (Colab). These notebooks demonstrate various big data processing and visualisation techniques using Apache Spark and other data science libraries in a cloud-based environment.

## Getting Started

To use these notebooks:

1. **Open Google Colab**: Visit [colab.research.google.com](https://colab.research.google.com)
2. **Load from GitHub**: Use the "File > Open notebook" option and select "GitHub"
3. **Enter Repository URL**: Use this repository's URL to access the notebooks
4. **Select Notebook**: Choose the desired notebook from the list
